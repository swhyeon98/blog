이 저장소는 블로그 글을 작성하고 관리하기 위한 공간입니다.

짧은 메모에서 시작해 초안을 만들고, 예제 코드와 함께 내용을 보완한 뒤  
완성된 글은 티스토리에 게시하는 방식으로 운영합니다.

## 다루는 내용

주로 아래 주제를 다룹니다.

- Java
- Spring Boot
- JPA / Hibernate
- Database
- Testing
- Troubleshooting
- Backend Architecture

## 글 작성 흐름

이 저장소의 문서는 아래 흐름으로 관리합니다.

1. `drafts/`에 아이디어 또는 초안 작성
2. `examples/`에 관련 예제 코드 작성
3. 글 구조 정리 및 보완
4. 게시 가능한 수준이 되면 `posts/`로 이동
5. 티스토리에 게시 후 상태를 `published`로 변경

## 디렉토리 구조

```text
.
├─ drafts/         # 메모, 아이디어, 초안
├─ posts/          # 게시 가능한 수준으로 정리된 글
├─ examples/       # 블로그 글에 포함되는 예제 코드
├─ assets/         # 이미지, 다이어그램, 첨부 자료
└─ README.md
```
## 작성 상태

문서는 아래 상태로 구분합니다.

- `idea`: 아이디어 메모
- `draft`: 초안 작성 중
- `review`: 검토 중
- `published`: 게시 완료

## 참고 사항

- 예제 코드는 학습 및 설명을 목적으로 작성되었습니다.
- 일부 문서는 작성 중인 초안일 수 있으며, 내용이 변경될 수 있습니다.
- 운영 환경에 적용할 때는 별도의 검토와 검증이 필요합니다.
